## 📖 Como Usar Este Repositório

### Navegando pelos Exercícios

Cada módulo possui:

- 📄 **README.md** - Documentação detalhada do exercício
- 💻 **Código-fonte** - Implementação completa
- 🎨 **Assets** - Imagens e recursos (quando aplicável)

### Executando os Projetos Localmente

#### Projetos HTML/CSS/JS Simples

```bash
# Navegue até o módulo ou projeto desejado
cd "Exercício Módulo X"
# ou
cd Project-4

# Abra o index.html no navegador ou use um servidor local
python3 -m http.server 8000
# ou
npx http-server -p 8000
```

#### Projetos com Build Tools

**Gulp (Módulo 16):**

```bash
cd "Exercício Módulo 16"
npm install
npm run dev
```

**Grunt (Módulo 18):**

```bash
cd "Exercício Módulo 18"
npm install
npm run dev
```

**SASS (Módulos 15, 19):**

```bash
cd "Exercício Módulo X"
npm install
npm run sass
```
